# Generator

함수의 실행을 중간에 멈췄다가 재개할 수 있는 기능이다. 필요한 순간에만 값ㄴ.. 사용...

# 사용 이유

기존 프로퍼티에 영향을 주지 않는 원시 타입으로, 생성된 심볼마다 고유한 키가 ~된다.

# 정의 및 형식

function* func() {
  yield 1;
  yield 2;
  yield 3;

  return "finished";
}

const var = func();

{
  value: yield 키워드 우측 값 || undefined,
  done: 모두 돌았는지 여부
}
