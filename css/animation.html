# transition

transition 속성들에 대한 축약형 속성이다. transition-property 속성에 명시한 속성의 값이 변한 만큼 애니메이션을 실행시킨다.

selector {
  /**
    * 1. transition-property
    * 2. transition-duration
    * 3. transition-timing-function
    * 4. transition-delay
   **/
  transition: 속성 | 애니메이션 지속 시간 | 타이밍 방식(전이 방식) | 지연 시간;
}

모든 속성은 생략이 가능하다. 단, 시간을 나타내는 값은 순서에 주의해야 한다! 첫 번째 시간은 무조건 duration 값, 두 번째는 delay 값으로 인식한다. (시간 단위: s)
