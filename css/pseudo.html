    # 의사 클래스와 의사 요소

    의사 클래스와 의사 요소에서 헷갈리는 몇 가지만 정리해본다!

    • 의사 클래스(pseudo class) - 선택된 대상에 상태별 style을 적용시키기 위한 것
    • 의사 요소(pseudo element) - 선택된 대상의 특정 부분만 선택하기 위한 것

    # 의사 클래스(psedo class)

    선택된 대상에 상태별 style을 적용시키기 위한 것으로, 동적 / 상태 / 구조 의사 클랙스가 있다.

    선택자:의사클래스 {
  .. style ..
}

    # 동적 의사 클래스 - :focus

    :focus는 포커스 역할을 가지고 있는 input 요소를 선택하는 동적 의사 클래스이다.

    # 상태 의사 클래스 - :enabled, :disabled

    의사 클래스 설명
    :enabled 사용할 수 있는 input 요소를 모두 선택
    :disabled 사용할 수 없는 input 요소를 모두 선택

    # 구조 의사 클래스

    의사 클래스 설명
    :nth-last-child(num) 뒤에서 n번째에 위치하는 자식 선택
    :only-child 자식 요소를 단 하나만 가지는 요소의 자식 선택
    :empty 자식 요소를 가지고 있지 않는 요소의 선택
    :root 문서의 root 요소 선택

    fist-child와 first-of-type의 차이
